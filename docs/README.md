## The game
Speed Dodger is a 2D game made by Ayaan.<br>
You dodge bad bullets and catch good bullets.<br>
They look exactly the same, you tell bullets apart based on speed alone.<br>
Good bullets are slow and bad bullets are fast.

## Quick Start
Open `index.html` in any modern browser,<br>
or use Live Server if you want to code and run.
